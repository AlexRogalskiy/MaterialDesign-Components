# `<mdi-input-select>`

The `mdi-input-select` component creates an select drop down.

```typescript
import '@mdi/web-components/mdiInputSelect';
import { MdiInputSelect } '@mdi/web-components/mdiInputSelect';
```

```html
<mdi-input-select></mdi-input-select>
```
